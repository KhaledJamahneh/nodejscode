┌──────────────────────────────────────────────────────────────────────────────┐
│ 📊 HOW IT WORKS                                                              │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│ Formula:                                                                     │
│   gallons_on_hand = current + delivered - returned_empties                  │
│                                                                              │
│ Example:                                                                     │
│   Current:  20 gallons on hand                                              │
│   Deliver:  100 gallons                                                     │
│   Return:   15 empty gallons                                                │
│   Result:   20 + 100 - 15 = 105 gallons on hand                             │
│                                                                              │
│ This tracks the net gallons currently at the client's location.             │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
